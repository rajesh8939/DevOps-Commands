To pull the files form github to Local machine

1. git clone <URL>

To push files form local to github

1. git config --global user.email "rajeshg@git.com"
2. git config --global user.name "rajesh"
3. git init
4. git add .
5. git commit -m "---"
6. git push -u origin master/main

To create branch commands

1. git status
2. git checkout -b branch-name
3. git checkout -b feature/new-feature
4. git branch
5. git push -u origin branch-name
